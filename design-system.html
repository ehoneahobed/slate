Principles

The five ideas behind every pixel.

When in doubt, choose the option that makes Slate feel more like a trusted book, and less like software.

01

Paper first

Warm, slightly-textured cream surface; ink for content. Chrome is the accent — never the star.

02

Ink is sacred

Everything the user drew is foreground. Our UI fades back — glass bars, soft borders, muted ink tones.

03

Calm color

Single warm terracotta accent. Use it for state, not decoration. One accent on screen at a time.

04

Type as ritual

Serif titles, clean sans UI, hand for annotations. Three voices, always in that order of importance.

05

Quiet interaction

Short, gentle motion. No bounce. No confetti. Respect prefers-reduced-motion.

06

A11y is non-negotiable

WCAG AA for text, 3:1 for chrome. Focus rings are more visible, not less. Keyboard-first.

Typography

Three voices. Always in order.

For engineers

Exported tokens.

Drop into app/globals.css. Also available as a JSON (ask in #design).

/* globals.css — Slate v1 */
:root {
  /* color · surface */
  --bg: #efe8d9;
  --paper: #f8f4ec;
  --paper-2: #f3ecdc;
  --chrome: #fbf8f1;
  --chrome-2: #efe8d5;
  --chrome-b: #dfd4b5;

  /* color · ink */
  --ink: #1f1c15;
  --ink-2: #4a4436;
  --ink-3: #7a7360;
  --ink-4: #ada48c;

  /* color · accent + state */
  --accent: oklch(0.58 0.13 45);
  --accent-2: oklch(0.72 0.10 45);
  --accent-soft: oklch(0.92 0.04 45);
  --ok: oklch(0.58 0.12 145);
  --warn: oklch(0.72 0.13 85);
  --danger: oklch(0.55 0.17 25);

  /* type */
  --ui: "Inter", system-ui, sans-serif;
  --serif: "Instrument Serif", Georgia, serif;
  --mono: "JetBrains Mono", ui-monospace, monospace;
  --hand: "Caveat", cursive;

  /* radius */
  --r-sm: 6px; --r-md: 10px;
  --r-lg: 14px; --r-xl: 22px;
  --r-full: 999px;

  /* space (4-based) */
  --space-1: 4px; --space-2: 8px;
  --space-3: 12px; --space-4: 16px;
  --space-5: 20px; --space-6: 24px;
  --space-8: 32px; --space-10: 40px;
  --space-12: 48px; --space-16: 64px;

  /* shadow · warm, not grey */
  --shadow-1: 0 1px 0 rgba(50,40,20,.04), 0 1px 2px rgba(50,40,20,.06);
  --shadow-2: 0 1px 0 rgba(50,40,20,.05), 0 6px 22px -8px rgba(50,40,20,.22);
  --shadow-3: 0 10px 40px -10px rgba(40,30,10,.25), 0 2px 8px rgba(40,30,10,.10);

  /* motion */
  --d-xs: 80ms; --d-sm: 120ms;
  --d-md: 180ms; --d-lg: 260ms;
  --ease-out: cubic-bezier(0.2, 0.8, 0.2, 1);
}
